# MetroWordle Helper - 智能概率辅助插件

## 项目概述

MetroWordle Helper 是一款专为 [MetroWordle.fun](https://metrowordle.fun) 游戏设计的智能辅助插件，基于信息熵与概率模型，为玩家提供最优的猜词推荐。插件通过科学的算法分析，帮助玩家在最短的步数内完成游戏。

## 核心功能

### 🧠 智能推荐算法
- 基于信息熵与最小期望剩余候选数（ERC）模型
- 每轮推荐最优猜测词，科学提升成功率

### 👥 对战模式优化
- 支持双人对战模式
- 利用对手反馈优化概率模型，显著提升对战胜率（+40%+）

### 🔄 跨设备同步
- 一键导出/导入 localStorage
- 游戏进度与统计数据永不丢失

### 📚 历史学习
- 自动记录每道题的答案历史
- 在后续游戏中提升相关词汇概率，越用越聪明

### ⌨️ 便捷输入
- 可视化手动输入系统
- 支持点击方块切换颜色，无需记忆复杂编码

### 📊 动态概率模型
- 基于历史猜词反馈实时更新候选词概率
- 结合词频与同音规则精准过滤

## 安装指南

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展（支持 Chrome / Edge / Firefox）
2. 访问项目下载页面安装脚本
3. 打开 [MetroWordle.fun](https://metrowordle.fun) 即可使用

## 使用方法

1. 在 MetroWordle 游戏中，插件会自动显示推荐词
2. 根据推荐词进行猜测
3. 将游戏结果（灰/绿/黄/蓝）输入插件
4. 插件会根据反馈更新概率模型并推荐下一个最优词

## 技术特点

- **科学算法**：采用信息熵和ERC模型，确保推荐的准确性
- **实时更新**：根据玩家输入实时调整候选词概率
- **用户友好**：直观的界面设计，操作简单便捷
- **数据安全**：所有数据仅保存在本地，绝不上传服务器
- **持续优化**：基于新的反馈不断调整概率模型

## 常见问题

**Q: 插件会收集我的数据吗？**
A: 不会！所有数据仅保存在你的浏览器 localStorage 中，绝不上传服务器，完全离线运行。

**Q: 为什么推荐词和我猜的不一样？**
A: 插件默认使用最优策略（基于信息熵）。如果您输入了非推荐词，插件会弹出"手动输入"面板，请按实际反馈填写以保持准确性。

**Q: 如何同步多台设备的数据？**
A: 在插件界面点击"导出数据"按钮，保存 JSON 文件；在另一台设备上点击"导入数据"即可实现跨设备同步。
